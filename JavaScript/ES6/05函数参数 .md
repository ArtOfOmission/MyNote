# 函数参数
## 1. 参数扩展/展开

### 1.1	收集剩余的参数
```javascript
function show(a, b, ...args) { }
//*rest parameter必须是最后一个
```

### 1.2 展开数组
```javascript
let arr = [1, 2, 3];
show2(...arr);
function show2(a, b, c) {
    console.log(a);
    console.log(b);
    console.log(c);
}

let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let arr3 = [...arr1, ...arr2];
```
展开后的效果，就跟直接把数组的内容拿出来一样。


## 2. 默认参数
```javascript
function show3(a = 4, b = 2, c = 3) {
    console.log(a);
    console.log(b);
    console.log(c);
}
show3(1);
```
